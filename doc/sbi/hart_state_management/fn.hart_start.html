Function hart_start

pub fn hart_start(
    hart_id: usize,
    start_addr: usize,
    private: usize,
) -> Result<(), SbiError>
Expand description

Start the specific hart ID at the given physical address along with a user-defined value. On success, the hart begins execution at the physical address with the hart ID in a0 and the user-defined value in a1, all other register values are in an undefined state.

ยงPossible errors

SbiError::InvalidAddress: start_address is an invalid address because it is either an invalid physical address or execution is prohibited by physical memory protection.

SbiError::InvalidParameter: The specified hart ID is either not valid or cannot be started in S-mode.

SbiError::AlreadyAvailable: The specified hart ID is already started.

SbiError::Failed: Start request failed for unknown reasons.
