Function remote_hfence_gvma

pub fn remote_hfence_gvma(
    hart_mask: HartMask,
    start_addr: usize,
    size: usize,
) -> Result<(), SbiError>
Expand description

Instructs the given harts to execute a HFENCE.GVMA for the region contained by start_addr and size. Only valid on harts which support the hypervisor extension. size is the size in bytes of the memory region for which an HFENCE.GVMA will be executed.

ยงPossible errors

SbiError::NotSupported: The function is either unimplemented or the target harts do not implement the hypervisor extension.
